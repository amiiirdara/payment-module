# Automoby — Order Module (Code Review Task)

این ریپو شامل ماژول سفارش و پرداخت بک‌اند اتوموبی است (NestJS + TypeORM) که برای تسک بررسی کد در اختیار شما قرار گرفته.

## نقاط ورود اصلی

- **`proceedCheckout`** — ‏`src/order/order.controller.ts` (خط ~251) → ‏`order.service.ts`
  ثبت نهایی سبد خرید و ساخت سفارش و اتصال به درگاه پرداخت.
- **`verifyOrder`** — ‏`src/order/order.service.ts` (خط ~1899)
  کال‌بک تأیید پرداخت؛ از کنترلر برای همه‌ی درگاه‌ها (زرین‌پال، اسنپ‌پی، ترب‌پی، سامان، تارا، اوانو، دیجی‌پی، ازکی، وایب و…) صدا زده می‌شود.

نمودار ERD ماژول در [`src/order/docs/order-erd.md`](src/order/docs/order-erd.md) موجود است (از روی entityهای فعلی تولید شده و گیت‌لب آن را رندر می‌کند).

## نکته‌ها

- فقط ماژول `order` در این ریپو قرار دارد؛ ماژول‌های بیرونی (payments، discount، search و…) عمداً حذف شده‌اند. بنابراین **کد کامپایل نمی‌شود و لازم هم نیست** — فراخوانی به سرویس‌های بیرونی را جعبه‌سیاه در نظر بگیرید.
- کد واقعی و در حال اجرا در production است؛ همان‌طور که هست بررسی‌اش کنید.

## خروجی مورد انتظار

طبق متن تسکی که جداگانه برایتان ارسال شده. خلاصه: نقاط قوت و ضعف، ریسک‌ها و گلوگاه‌ها، و اینکه اگر مالک این کد بودید اولین تغییری که می‌دادید چه بود، چرا و چگونه.
